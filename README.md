# Pet Saúde — Site Institucional
Site desenvolvido como projeto de portfólio por Diogo Falcão (FalAiquoc)

🏪 Sobre o Negócio
O Pet Saúde é um completo centro de cuidados animais localizado no bairro de Lagoa Nova em Natal/RN. Reúne atendimento veterinário de excelência, farmácia pet especializada e serviços de banho e tosa premium focados no bem-estar de cães e gatos.

🛠️ Tecnologias
- Vite + React + TypeScript
- Tailwind CSS
- Lucide React

📐 Referências de Design
- https://github.com/adrianhajdin/project_hoobank — Referência para layout de landing page responsiva e moderna.
- https://github.com/WebDevSimplified/react-shopping-cart — Padrões de arquitetura de catálogo e exibição de cards.
